VoCore
======

Install OpenWrt build prerequisites (Ubuntu 14.04):
```sh
sudo apt-get install git-core build-essential libssl-dev libncurses5-dev unzip gawk subversion mercurial
```

Build OpenWrt:
```sh
sh build-openwrt.sh
```
